# Project Images

This directory contains images and assets for the project documentation.

## Banner Image

Place your project banner/logo image here:
- **Recommended name**: `ocpp-emulator-banner.png`
- **Recommended size**: 1200x400px or similar aspect ratio
- **Format**: PNG or SVG (PNG preferred for compatibility)

## Usage

The banner image is referenced in the main `README.md`:

```markdown
![OCPP Emulator](docs/assets/images/ocpp-emulator-banner.png)
```

## Image Suggestions

You can create or use:
1. **Screenshot** of the web UI dashboard
2. **Architecture diagram** showing the system components
3. **Logo** with project name
4. **Banner** with key features highlighted

## Tools for Creating Images

- **Screenshots**: Use your OS screenshot tool or browser dev tools
- **Diagrams**: Draw.io, Excalidraw, or Mermaid
- **Graphics**: Canva, Figma, or GIMP
- **Architecture**: Use tools like PlantUML or diagrams.net

## Alternative: Using External Images

If you prefer to host images externally (e.g., GitHub releases, CDN), you can use absolute URLs:

```markdown
![OCPP Emulator](https://raw.githubusercontent.com/yourusername/ocpp-emu/main/docs/assets/images/ocpp-emulator-banner.png)
```
